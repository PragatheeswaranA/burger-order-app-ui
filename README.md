# Burger Order App 🍔

## Overview
This is a UI/UX design project for a food delivery mobile app designed in Figma.

## Problem
Users struggle with quick food ordering and tracking.

## Solution
A clean and intuitive UI with fast checkout and smooth navigation.

## Features
- Easy food browsing
- Quick checkout
- Order tracking

## Tools Used
- Figma
